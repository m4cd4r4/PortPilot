## ğŸ”„ Feature #2: MCP Auto-Refresh

### **What It Does:**
**No more manual refresh!** When MCP adds/updates/deletes apps, PortPilot automatically detects the change and refreshes the UI.

### **How It Works:**
1. **File watcher** monitors `portpilot-config.json` for changes
2. **Debounced reload** (100ms) prevents rapid duplicate reloads
3. **Auto-refresh UI** - Renderer receives event and reloads apps
4. **Toast notification** - "Apps list updated (external change detected)"

### **User Experience:**
```
Before v1.6:
  MCP adds app â†’ User clicks Refresh button â†’ App appears

After v1.6:
  MCP adds app â†’ App appears automatically + toast âœ¨
```
